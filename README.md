## simple-crud-react ##
Simple CRUD Operations UI using ReactJS &amp; Material UI & PrimeReact


**I. Install & Start Json-Server**

`npm i json-server`

`json-server --watch public/data/db.json --port 3001`

**II. Install & Start React App**

`npm install`

`npm start`
